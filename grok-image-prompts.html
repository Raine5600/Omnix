BONUS ★

Tips for GIF Consistency Across All 6 Frames

Keep these consistent across every Grok generation to make the frames look like they belong together.

Pro Tips
🎨 Add to EVERY prompt: "photorealistic, 4K, cinematic lighting, sharp focus, professional photography" — this keeps the visual style unified.
🎨 Color consistency: Always mention "dark navy blue #1A2340 and gold/amber" when the website UI is visible so Grok keeps the brand colors consistent.
🎨 Same person across frames: If Grok supports "seed" or "style reference," use the same seed number for Frames 2, 3, 4, and 6 to keep the same customer/hands consistent throughout.
🎨 GIF assembly order: Frame 1 (3s) → Frame 2 (2s) → Frame 3 (2s) → Frame 4 (1.5s) → Frame 5 (2.5s) → Frame 6 (3s). Loop once or twice.
